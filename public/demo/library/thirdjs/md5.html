LarryMS框架扩展之md5/base64组件【提供于标准版及以上】

md5/base64组件：这是LarryMS框架中提供封装的md5和base64两个组件（来自github），MD5的全称是Message-Digest Algorithm 5（信息-摘要算法）；这两个组件使用场景较多，开发中也非常实用，兼容Node.js等服务器端环境，RequireJS，Browserify或webpack等模块加载器以及所有Web浏览器。在前端使用md5加密好处很多这里不再赘述、base64也会经常用到，下面就对md5/base64组件的使用做下示例介绍：

larryms官方社区提供所有所提供扩展组件的维护和更新。当前版本：md5 v1.2.1/base64 v0.0.3
MD5加密示例【效果示例】
执行MD5加密
MD5加密【Code示例】 
layui.use(['md5'],function(){
    var md5 = layui.md5;
    $('#md5Btn').on('click',function(){
        var input = $('#input1').val();
        $('#output1').text($.md5(input));
    });
});
//js前端nd5加密与后台语言所执行的是一样的算法，一般简单场景加密是用md5在前端也可以处理
//以上代码段所示为larryms框架中md5组件的使用方式
Base64示例【效果示例】
执行Base64编码
Base64【Code示例】 
layui.use(['base64'],function(){
    var base64 = layui.base64;
    $('#base64Btn').on('click',function(){
        var input = $('#input2').val();
        $('#output2').text($.base64.encode(input));
    });
});
//以上代码段所示为larryms框架中base64组件的使用方式
//base64 有encode和decode，并且base64可以编码文件，如常见的对图片转化为base64编码保存
